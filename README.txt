# Tarea-2-EDD
En el zip se encuentra: 
-el archivo DeBlock.cpp, el cual corresponde al TDA DeBlock requerido en la tarea
-Archivo test.cpp el cual se encarga de correr la prueba requerida de utilización del TDA DeBlock
-el archivo nombres.txt, con los nombres de cada integrante y qué realizó en la tarea.
-el archivo pruebas.txt utilizado para probar el TDA DeBlock
-el archivo README.txt, con lo solicitado.
-el archivo tLista.cpp, con la implementación de un TDA de listas basadas en punteros para el correcto funcionamiento de DeBlock
-Archivo tListadeLista.cpp con la implementación de un TDA de Listas de Listas basada en arreglos para el correcto funcionamiento de DeBlock


El archivo test.cpp se encarga de realizar lo requerido al correr el programa en la entrada con el archivo "pruebas.txt"

Compiladas con el comando g++ archivo.cpp -o output -Wall, en g++ de Ubuntu.
